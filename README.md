<h1 align="center">AVL tree</h1>

## Description

 Implementation of the AVL tree.

## How to use

1. Clone <br>
    write <code>git clone https://github.com/baitim/AVLTree.git</code> in terminal

2. Go to folder <br>
    write <code>cd AVLTree</code> in terminal

3. Make dir for objects <br>
    write <code>mkdir obj</code> in terminal

4. Build <br>
    write <code>make</code> in terminal

5. Run <br>
    write <code>./med</code> in terminal
